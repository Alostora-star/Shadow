# بوت متجر الخدمات الرقمية - النسخة المحسنة

هذه هي النسخة المحسنة من بوت تليجرام لمتجر الخدمات الرقمية، والتي تتضمن تحسينات كبيرة في الأمان، إدارة البيانات، وقابلية التوسع.

## التحسينات الرئيسية

1.  **تحسين الأمان (Enhanced Security)**:
    *   تم فصل البيانات الحساسة (مثل توكن البوت) عن الكود. يتم الآن تحميلها من ملف `.env` الذي يجب إنشاؤه محلياً.
    *   تم إضافة ملف `.gitignore` لمنع رفع الملفات الحساسة (مثل `.env` و `bot_data.db`) إلى مستودعات Git عن طريق الخطأ.

2.  **إدارة ديناميكية للمنتجات (Dynamic Product Management)**:
    *   تم نقل جميع بيانات المنتجات والفئات من ملف `data.py` الثابت إلى قاعدة بيانات SQLite.
    *   يمكن الآن إضافة وتعديل وحذف المنتجات والفئات والسيرفرات دون الحاجة إلى تعديل الكود البرمجي.

3.  **لوحة تحكم للمسؤول (Admin Panel)**:
    *   تمت إضافة لوحة تحكم شاملة للمسؤول (يمكن الوصول إليها عبر الأمر `/admin`) لإدارة المتجر مباشرة من تليجرام.
    *   يمكن للمسؤول الآن:
        *   إضافة/تعديل/حذف الفئات.
        *   إضافة/تعديل/حذف المنتجات وتحديد أسعارها.
        *   عرض قائمة بجميع المنتجات والفئات.

4.  **تحسين تجربة المستخدم (Improved User Experience)**:
    *   **حفظ معرفات الألعاب**: يمكن للمستخدمين الآن حفظ معرفات الألعاب (Game IDs) الخاصة بهم لتسريع عمليات الشراء المستقبلية.
    *   عند الشراء، يتم عرض قائمة بالمعرفات المحفوظة للاختيار منها، مع خيار إدخال معرف جديد.

5.  **نظام دفع مرن (Flexible Payment System)**:
    *   تم إعادة هيكلة نظام الدفع في ملف `config.py` ليصبح أكثر مرونة، مما يسهل إضافة أو تعطيل طرق الدفع في المستقبل.

## كيفية الإعداد والتشغيل

1.  **تثبيت المكتبات المطلوبة**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **إعداد متغيرات البيئة**:
    *   قم بنسخ ملف `.env.example` إلى ملف جديد باسم `.env`.
    *   افتح ملف `.env` واملأ القيم المطلوبة:
        *   `BOT_TOKEN`: توكن البوت الخاص بك من BotFather.
        *   `ADMIN_USER_ID`: معرف تليجرام الخاص بك (يمكنك الحصول عليه من `@userinfobot`).
        *   `GROUP_ID` و `GROUP_JOIN_LINK` (اختياري): إذا كنت تريد إجبار المستخدمين على الانضمام إلى قناة.

3.  **تهيئة قاعدة البيانات وتشغيل البوت**:
    *   سيقوم البوت بإنشاء وتهيئة قاعدة البيانات (`bot_data.db`) تلقائياً عند أول تشغيل.
    *   قم بتشغيل البوت:
    ```bash
    python bot.py
    ```

## كيفية الاستخدام (للمسؤول)

1.  **الوصول إلى لوحة التحكم**: أرسل الأمر `/admin` إلى البوت.
2.  **إضافة فئة**: من لوحة التحكم، اختر "إضافة فئة" واتبع التعليمات.
3.  **إضافة منتج**: من لوحة التحكم، اختر "إضافة منتج" واتبع الخطوات لاختيار الاسم، السعر، الوصف، والفئة.
4.  **تأكيد الطلبات**: استخدم الأوامر `/admin_confirm_deposit` و `/admin_confirm_shipped` كما في النسخة السابقة.
